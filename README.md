Before you begin, ensure you have Node.js and Yarn installed.

1. **Install dependencies:**
   ```bash
   yarn install
   ```

2. **Run local Ethereum network:**
   ```bash
   yarn chain
   ```

3. **Deploy smart contracts:**
   ```bash
   yarn deploy
   ```

4. **Start the Next.js app:**
   ```bash
   yarn start
   ```
